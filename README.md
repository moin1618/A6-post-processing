# A6 Post-Processing Scripts

This repository contains the post-processing scripts used for processing CFD simulation results and generating selected plots/tables for the M.Tech thesis:

**Numerical Investigation of Forced-Convection-Dominated Heat Transfer in a Radiant-Wall Room with Displacement Ventilation**

Author: **Mir Moin Afzal**  
Programme: **M.Tech Thermal Engineering**  
Department: **Department of Mechanical Engineering**  
Institute: **National Institute of Technology Srinagar**

## Purpose

The scripts in this repository are provided to support the results presented in the thesis appendix and to improve reproducibility of the post-processing workflow.

They may include routines for:

- Reading exported CFD result data
- Processing temperature, velocity, and heat-transfer-related quantities
- Generating plots used in the thesis
- Preparing tables or processed values for analysis

## Repository Contents

The repository may contain:

- Post-processing scripts
- Input/example data files, if permitted
- Generated plots or output files, if required
- Notes related to running the scripts

Large raw simulation files and confidential/project-restricted files are not included.

## How to Use

1. Download or clone this repository.
2. Place the required exported CFD data files in the appropriate folder.
3. Run the post-processing scripts as described in the script comments or file names.
4. The processed outputs/plots can be used for comparison with the thesis results.

## Software Requirements

The scripts may require commonly used scientific computing tools such as:

- Python
- NumPy
- pandas
- Matplotlib

Specific requirements may depend on the individual script.

## Thesis Reference

These scripts correspond to the appendix section related to software/code used for obtaining plots and results in the thesis.
